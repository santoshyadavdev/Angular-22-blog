TypeScript 6.0 Features in Angular 22

Exploring the key changes in TypeScript's bridge release to the native Go-based TS 7.0

1. strict: true by Default

TypeScript 6.0 enables strict mode by default in new projects. Previously, you had to explicitly opt in to strict checking with "strict": true in your tsconfig. Now all strict family flags , strictNullChecks, strictFunctionTypes, strictBindCallApply, noImplicitAny, and more , are on unless you explicitly disable them.

Angular CLI projects already defaulted to strict: true, so this change mainly aligns the broader TypeScript ecosystem with Angular's existing best practice.
